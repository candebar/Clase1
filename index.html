Pasos para preparar un mate

  1. Poner yerba en el mate
  2. Sacudir un poco para sacarle el polvillo
  3. Agregar un poco de agua caliente
  4. Con la bombilla presionar un poco la yerba para un lado formando una "montañita"
  5. Colocar la bombilla en el mate y mas agua caliente para completar
  6. Listo!
